# 道具设计

## 目前有什么道具？

目前游戏只有 **1 种道具**——**散弹（S 道具）**！

---

## 散弹道具

### 长什么样子？

![道具样子]
- 蓝色菱形，中间有一个白色的 **"S"** 字母
- 周围有一圈淡淡的蓝光

### 怎么获得？

- 每当你**消灭一只敌人**（黄色圆圈或绿色箭头），它有 **5% 的概率**掉落一个散弹道具。也就是说，平均每打 20 只敌人才会出现 1 个。挺稀有的！
- 道具会从敌人死掉的地方出现，然后**慢慢往下掉**。
- 你的飞机碰到它，就可以捡起来。

### 怎么使用？

#### 捡到之后
散弹不会自动发射，而是**存起来**。屏幕左下角会出现一个"散弹"图标。

#### 激活方法（两种方式都可以）
| 平台     | 操作方法                     |
| -------- | ---------------------------- |
| 💻 电脑   | 按键盘上的 **数字 1** 键     |
| 📱 手机   | 点击屏幕上左下角的 **"散弹"图标** |

#### 激活后有什么效果？

激活后，你的子弹会从 **1 发变成 3 发**！
- 中间一发：白色，向前直射
- 左边一发：浅蓝色，向左偏 15 度
- 右边一发：浅蓝色，向右偏 15 度

这样一次射击能打中更多敌人，打 BOSS 也更快！

### 持续多久？

散弹效果持续 **5 秒**（5000 毫秒），时间到了就变回普通单发子弹。

### 注意事项

| 情况                 | 会怎样？                         |
| -------------------- | -------------------------------- |
| 已经有一个散弹存着   | 再捡新的没用，不能叠加           |
| 正在使用散弹         | 再捡新的没用，时间不会加长       |
| 散弹效果消失         | 如果包里还有一个，可以再次激活   |

# 关卡设计

## 总共有几关？

目前游戏有 **2 关**。打通第 2 关就通关啦！

---

## 第 1 关：测试关

### 敌人阵容

| 敌人种类     | 数量 | 说明                   |
| ------------ | ---- | ---------------------- |
| 🟡 黄色圆圈   | 10 只 | 会往下飞，还会朝你开火 |
| 🟢 绿色箭头   | 5 只  | 会追着你跑             |

- 每次场上只出现 **1 只**敌人，打完一只才出下一只。
- 全部 15 只打完，过关！

### 过关条件

消灭所有 15 只敌人。

---

## 第 2 关：boss 来了

### 敌人阵容

| 敌人种类       | 数量 | 说明                   |
| -------------- | ---- | ---------------------- |
| 🟡 黄色圆圈     | 5 只  | 和第一关一样           |
| 🟢 绿色箭头     | 10 只 | 比第一关更多！         |
| 🔴 BOSS（大六边形） | 1 只  | 血超厚，子弹会追踪你！ |

- 每次场上出现 **2 只**敌人，打得更快。
- 普通敌人全部消灭后，**BOSS 才登场**！
- 打败 BOSS 才算过关。

### 过关条件

先打完 15 只普通敌人，再打败 BOSS。

---

## 关卡过渡动画

打完一关之后，不会马上进入下一关，而是会播放一段小动画：

1. 你的飞机飞到屏幕正中间
2. 飞机向上飞出屏幕
3. 屏幕变黑，显示出下一关的名字
4. 几秒后自动进入新关卡

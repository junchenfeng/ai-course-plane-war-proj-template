# 数值设置

这里记录了你（玩家）和敌人、BOSS 的各种"属性数值"。简单说，就是**谁有多快、谁有多硬、谁伤害多高**。

---

## 👦 玩家（你）

### 生命值

| 项目     | 数值 | 说明                       |
| -------- | ---- | -------------------------- |
| 最大生命 | 5 点  | 你有 5 颗红心 ♥♥♥♥♥        |
| 受伤     | 扣 1 点 | 被敌人或敌人子弹碰到就扣 1 |
| 死亡     | 0 点  | 生命归零，游戏结束         |

### 移动

| 项目     | 数值 | 说明                       |
| -------- | ---- | -------------------------- |
| 移动速度 | 5    | 每帧移动 5 像素，节奏适中  |

### 射击

| 项目         | 数值    | 说明                         |
| ------------ | ------- | ---------------------------- |
| 子弹速度     | 8       | 子弹向上飞得挺快             |
| 射击间隔     | 150 毫秒 | 大约每秒能打 6-7 发          |
| 散弹角度     | 15 度   | 左右两发散弹各偏 15 度       |
| 散弹持续时间 | 5000 毫秒 | 拾取后激活，持续 5 秒       |

### 无敌保护

| 项目       | 数值    | 说明                                     |
| ---------- | ------- | ---------------------------------------- |
| 受伤后无敌 | 500 毫秒 | 受伤后有半秒的无敌时间，不会被连续伤害   |
| 无敌时外观 | 半透明  | 飞机变半透明，告诉你现在是安全的         |

## 📐 画布大小

游戏默认画布是 **800 × 600** 像素，但实际打开时会**自动铺满你的屏幕**（浏览器窗口有多大就多大），所以每个玩家的画面尺寸可能不一样。

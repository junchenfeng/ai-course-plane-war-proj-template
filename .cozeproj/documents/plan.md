# 道具状态变更实现计划

## 概述

为飞行射击游戏的道具指示器添加状态视觉反馈：无道具时显示灰色，持有道具时显示亮色，使用道具时显示亮色闪烁效果。

## 技术方案

| 维度 | 选择 | 理由 |
|------|------|------|
| 修改范围 | CSS 样式文件 | 只需要调整样式，无需改动逻辑 |
| 动画方式 | CSS @keyframes | 原生 CSS 动画性能好，实现简单 |
| 兼容性 | 保持现状 | 项目已使用现代 CSS，无需降级 |

## 功能模块

### 样式模块

- **职责**：定义道具指示器三种状态的视觉样式
- **要点**：
  - 无道具：灰色（#555），无光效
  - 持有道具：蓝色（#4488ff），脉冲光效
  - 使用中：绿色（#44ff44），快速闪烁效果

## 是否有原型设计

否

## 实施步骤

1. 分析当前道具指示器样式状态
2. 修改 CSS 样式文件，为 active 状态添加闪烁动画
3. 测试验证三种状态的视觉效果

## 页面规格

不涉及页面改动，仅调整现有 UI 元素的样式。
